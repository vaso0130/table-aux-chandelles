# 燭光牌桌 · La Table aux Chandelles

單檔網頁版占卜牌桌,內含三副牌組:雷諾曼(Petit Lenormand,36 張)、西碧拉(Vera Sibilla Italiana,52 張、正逆位)、貝林神諭(L'Oracle Belline,53 張、正逆位)。直接用瀏覽器開啟 `index.html` 即可,無需伺服器。

## 牌組

- **雷諾曼**:36 張,八種盤面——單卡指引、三卡時間之流、五卡敘事之線(可再補兩張澄清牌)、五卡十字陣、十卡金字塔(1-2-3-4 四層)、九宮格、十二宮位生命之輪(占星宮位對應)、大盤 Grand Tableau(含 36 宮位對照與指示牌落點)。
- **西碧拉**:52 張(四花色各 13 張,對應整副撲克牌),每張抽出時由命運決定正位或逆位,牌義各自不同。五種盤面——單卡今日低語、三卡時間之流、五卡十字陣、九宮格、十五卡大陣(三列五欄:過去/現在/未來)。
- **貝林神諭**:53 張——命運、男性之星、女性之星三張主牌,七星體(太陽/月亮/水星/金星/火星/木星/土星)各七張,加無編號的藍卡守護牌;正逆位。五種盤面——單卡今日神諭、三卡時間之流、五卡十字陣、七卡七星陣(七星體領域掃描)、九宮格。

## 牌義字典

頁面底部的「牌義字典」依目前牌組列出所有牌;點開任一張可學習:定義與關鍵詞、正/逆位牌義、對應(撲克/星體家族)、雷諾曼的組合語法(主詞/謂語用法與大盤宮位含義),以及同一張牌落在八種常見位置(過去/現在/未來/核心/意識/潛藏/建議/結果)時的讀法示範。

## 卡背

三副牌各有專屬卡背:雷諾曼(深藍金紋)、西碧拉(酒紅金紋・四花色徽)、貝林(靛紫金紋・七星環徽),切換牌組時自動更換。

## 功能

- **真實洗牌與切牌**:牌組從出廠順序開始,每按一次「洗牌」執行一次 GSR 模型鴿尾式交錯洗牌(`crypto` 隨機,建議七次以上),「切牌」由使用者點選位置真實切開牌疊;沒洗過牌不能開始抽。
- **親手抽牌**:牌堆扇形展開,由使用者逐張點選。九張以上的盤面可「讓命運抽完剩下的牌」。
- **解讀帳冊**:逐位置牌義(西碧拉依正逆位切換)、關鍵詞、雷諾曼牌與牌之間的組合語句;點牌面可跳至對應解讀。
- **你的問題**:牌桌上方可輸入問題,會自動寫進 AI 提示詞並隨紀錄保存。
- **牌義跟著牌走**:滑鼠移過(或手機點按)牌面,牌旁浮出位置意義+牌義+撲克對應;大盤含 36 宮位組合語義。
- **AI 解讀 Prompt**:抽完後一鍵產生完整提示詞(牌組、盤面、位置、牌名、正逆位、撲克對應、關鍵詞、你的問題),由使用者自行複製貼給任何 AI。
- **占卜歷史**:每次完成自動存入 `localStorage`(上限 50 筆,含牌組與正逆位),可重看(還原整個牌陣與解讀)、刪除、清空。

## 牌面

牌面依十九世紀鍍金石版畫式樣重繪,每張保留傳統撲克牌對應符號(如騎士=紅心9、西碧拉鴻運=梅花5)與經典意象。缺圖時,網頁自動退回符號版牌面。

開發測試:`index.html?rwdtest=盤面id`(如 `nine`、`sib-fifteen`、`bel-seven`)會自動選盤、洗牌並抽滿整個牌陣;`index.html?dicttest=牌序&deck=牌組id` 直接打開字典詳細頁。
