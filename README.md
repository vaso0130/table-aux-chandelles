## 功能

- **真實洗牌與切牌**:GSR 鴿尾式交錯洗牌(`crypto` 隨機)、點選位置真實切牌;易經擲卦盤面改為擲錢起卦。
- **親手抽牌**:牌堆扇形展開逐張點選;九張以上盤面可「讓命運抽完剩下的牌」。
- **解讀帳冊**:逐位置牌義(依正逆位切換)、關鍵詞、雷諾曼組合語句;點牌面彈出牌義浮框。
- **AI 解讀 Prompt**:一鍵產生完整提示詞(牌組、盤面、位置、正逆位、對應、六爻紀錄、你的問題與語氣),複製貼給任何 AI。
- **牌義字典**(獨立分頁):依牌組瀏覽全部牌,含正逆位牌義、對應、雷諾曼組合語法與大盤宮位、八種常見位置的讀法示範。
- **占卜歷史**:自動存入 `localStorage`(上限 50 筆,含牌組/正逆位/起卦紀錄),可重看、刪除、清空。
- **專屬卡背**:九副牌各一款,切換牌組自動更換。

## 牌面

西方館依十九世紀鍍金石版畫式樣重繪(象牙緞紋底、洛可可金框、頂部徽章);東方館為朱紅灑金・工筆重彩式樣(雲紋回紋金框、篆印式徽章)。缺圖時網頁自動退回符號版牌面。

開發測試:`index.html?rwdtest=盤面id`(如 `nine`、`sib-fifteen`、`yi-cast`)自動完成整個牌陣;`index.html?dicttest=牌序&deck=牌組id` 直接打開字典詳細頁;別館 `astro.html?test=盤式id`、`numbers.html?test=題目id`、`bazi.html?test=命術id`、`shi.html?test=式id`、`mix.html?test=id1,id2,id3` 以樣本資料自動起盤(成功時網頁標題為 *-TEST-OK)。
